\## 📖 Описание



\*\*NF\_Void Terminal\*\* — это полнофункциональный десктопный терминал, созданный на базе \*\*Electron.js\*\*. Приложение предоставляет реальный доступ к файловой системе вашего компьютера, сетевым инструментам и функциям безопасности, всё в стильном интерфейсе в стиле классического терминала.



Идеально подходит для:

\- 🔹 Системных администраторов

\- 🔹 Специалистов по кибербезопасности

\- 🔹 Разработчиков

\- 🔹 Любителей командной строки



---



\## ✨ Возможности



\### 📁 Работа с файлами

| Функция | Описание |

|---------|----------|

| `ls`, `cd`, `pwd` | Навигация по директориям |

| `cat`, `touch`, `mkdir` | Чтение и создание файлов |

| `rm`, `cp`, `mv` | Управление файлами |

| `echo` | Вывод текста |



\### 🌐 Сетевые инструменты

| Функция | Описание |

|---------|----------|

| `port` | Проверка статуса порта |

| `nmap` / `scan` | Сканирование портов (1-1024) |

| `netinfo` | Информация о сетевых интерфейсах |

| `dns` | DNS запросы |

| `traceroute` | Трассировка маршрута |



\### 🛡️ Кибербезопасность

| Функция | Описание |

|---------|----------|

| `hash` | Генерация хешей (MD5, SHA1, SHA256, SHA512) |

| `passwd` | Генерация безопасных паролей |

| `secscan` / `audit` | Аудит файлов на уязвимости |

| `base64` | Кодирование/декодирование |

| `verify` | Проверка целостности файлов |

| `logs` | Просмотр логов безопасности |



\### 💻 Системные команды

| Функция | Описание |

|---------|----------|

| `ps` / `processes` | Список процессов |

| `kill` | Завершение процессов |

| `sysinfo` | Информация о системе |

| `top` | Мониторинг ресурсов |

| `exec` | Выполнение системных команд |

\## 📦 Установка



\### Требования

\- \*\*Node.js\*\* версии 18.x или выше

\- \*\*npm\*\* версии 9.x или выше



\### Шаги установки



1\. \*\*Клонируйте репозиторий:\*\*

```bash

git clone https://github.com/yourusername/nf-void-terminal.git

cd nf-void-terminal

```



2\. \*\*Установите зависимости:\*\*

```bash

npm install

```



3\. \*\*Запустите приложение:\*\*

```bash

npm start

```

\## 📚 Использование



\### Примеры команд



\#### 🔐 Генерация безопасного пароля

```bash

passwd 32 -s

```

\*Создаст пароль из 32 символов с символами\*



\#### 🔍 Сканирование портов

```bash

nmap localhost 1 1000

```

\*Просканирует порты с 1 по 1000\*



\#### 📄 Аудит безопасности файла

```bash

secscan config.json

```

\*Проверит файл на наличие паролей, API-ключей и других уязвимостей\*



\#### 🔐 Генерация хеша

```bash

hash sha256 mypassword123

```



\#### 📊 Информация о системе

```bash

sysinfo

```



\#### 🌐 DNS запрос

```bash

dns google.com

```

\## 🔒 Безопасность



\### Важные примечания



⚠️ \*\*Приложение имеет реальный доступ к вашей файловой системе!\*\*



\- Команды `rm`, `mkdir`, `touch` создают/удаляют файлы на вашем компьютере

\- Сканирование портов ограничено диапазоном 1-1024 для безопасности

\- Системные директории защищены от удаления

\- Все действия безопасности логируются в `~/.nf-void-logs/`



\### Рекомендации

1\. Не запускайте команды от имени администратора без необходимости

2\. Проверяйте команды перед выполнением

3\. Регулярно просматривайте логи безопасности
